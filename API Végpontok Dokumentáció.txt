# Duális Képzés Backend API Végpontok

Ez a dokumentum tartalmazza a rendszer összes elérhető API végpontját, funkcionális csoportokba rendezve.

## Hitelesítés (/api/auth)
- POST /api/auth/register: Új felhasználó regisztrációja a rendszerbe.
- POST /api/auth/login: Meglévő felhasználó bejelentkezése és token kérése.

## Statisztikák (/api/stats)
- GET /api/stats: Rendszerszintű statisztikák lekérése. Csak bejelentkezett felhasználók számára elérhető.

## Hallgatók (/api/students)
- GET /api/students: Az összes hallgató listázása.
- GET /api/students/me: A bejelentkezett hallgató saját profiljának lekérése.
- PUT /api/students/me: A bejelentkezett hallgató saját profiljának frissítése.
- DELETE /api/students/me: A bejelentkezett hallgató saját profiljának törlése.
- GET /api/students/:id: Egy specifikus hallgató adatainak lekérése azonosító (ID) alapján.
- PUT /api/students/:id: Egy specifikus hallgató adatainak frissítése azonosító (ID) alapján.
- DELETE /api/students/:id: Egy specifikus hallgató törlése azonosító (ID) alapján.

## Általános Felhasználói Műveletek (/api/users)
- GET /api/users/inactive: Az inaktív felhasználók listázása.
- PATCH /api/users/:id/reactivate: Egy inaktív felhasználó fiókjának újraaktiválása azonosító (ID) alapján.
- PATCH /api/users/:id/deactivate: Egy aktív felhasználó fiókjának deaktiválása azonosító (ID) alapján.

## Cégek (/api/companies)
- GET /api/companies/inactive: Az inaktív cégek listázása.
- GET /api/companies: Az összes cég listázása.
- POST /api/companies: Új cég létrehozása a rendszerben.
- GET /api/companies/:id: Egy specifikus cég adatainak lekérése azonosító (ID) alapján.
- PATCH /api/companies/:id: Egy specifikus cég adatainak frissítése azonosító (ID) alapján.
- DELETE /api/companies/:id: Egy specifikus cég törlése azonosító (ID) alapján.
- PATCH /api/companies/:id/reactivate: Egy inaktív cég újraaktiválása azonosító (ID) alapján.
- PATCH /api/companies/:id/deactivate: Egy aktív cég deaktiválása azonosító (ID) alapján.

## Állásajánlatok és Pozíciók (/api/jobs)
- GET /api/jobs/positions: Az összes elérhető pozíció listázása.
- GET /api/jobs/positions/:id: Egy specifikus pozíció részletes adatainak lekérése azonosító (ID) alapján.
- POST /api/jobs/positions: Új pozíció létrehozása (csak jogosult felhasználóknak).
- PATCH /api/jobs/positions/:id: Meglévő pozíció adatainak frissítése azonosító (ID) alapján.
- DELETE /api/jobs/positions/:id: Pozíció törlése azonosító (ID) alapján.
- PATCH /api/jobs/positions/:id/deactivate: Pozíció deaktiválása (kivonása a publikus listából) azonosító (ID) alapján.

## Hírek (/api/news)
- GET /api/news: Publikus hírek listázása a felhasználók számára.
- GET /api/news/:id: Egy specifikus hír részleteinek lekérése.
- POST /api/news/admin: Új hír létrehozása (csak adminoknak).
- GET /api/news/admin: Az összes hír listázása adminisztrátori nézetben.
- GET /api/news/admin/archived: Az archivált hírek listázása (csak adminoknak).
- GET /api/news/admin/:id: Egy specifikus hír adminisztrátori adatainak lekérése.
- PATCH /api/news/admin/:id: Hír tartalmának frissítése (csak adminoknak).
- PATCH /api/news/admin/:id/archive: Hír archiválása (csak adminoknak).
- PATCH /api/news/admin/:id/unarchive: Archivált hír visszaállítása (csak adminoknak).
- DELETE /api/news/admin/:id: Hír törlése (csak adminoknak).

## Jelentkezések (/api/applications)
- POST /api/applications: Jelentkezés leadása egy meghirdetett pozícióra (hallgatóknak).
- GET /api/applications: A bejelentkezett hallgató saját jelentkezéseinek listázása.
- PATCH /api/applications/:id/retract: Leadott jelentkezés visszavonása.
- GET /api/applications/company: A céghez beérkezett jelentkezések listázása (céges felhasználóknak).
- PATCH /api/applications/company/:id/evaluate: Jelentkezés értékelése/bírálata (céges felhasználóknak).
- PATCH /api/applications/company/:id: Meglévő értékelés frissítése.
- GET /api/applications/admin: Az összes rendszerbeli jelentkezés listázása (rendszeradminoknak).
- GET /api/applications/admin/:id: Egy specifikus jelentkezés részleteinek lekérése (rendszeradminoknak).
- PATCH /api/applications/admin/:id: Jelentkezés adatainak adminisztrátori módosítása.
